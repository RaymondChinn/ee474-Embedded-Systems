# ee474-Embedded-Systems
Personal portfolio of work from ee474 Embedded Systems

# Lab 3
Lab 3 focuses on serial communication using I2C, implementing non-preemptive scheduling algorithms, and the development and management of interrupts and ISRs for task handling.

# Lab 4
Lab 4 implements a shortest time remaining first preemptive algorithm, introduces parallel computing utilizing the dual-core architecture of the ESP32, and integrates these concepts using FreeRTOS.
